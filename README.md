## Exercice site de l'entreprise VéhiLoc

Pour utiliser ce projet :
## Installation locale
git clone https://github.com/zacube/p8Tasklinker.git

## Prérequis
Composer version 2.8.10

PHP version 8.2.12


## Configuration des variables d’environnement

Créer un fichier .env.local, et renseigner les valeurs suivantes :

APP_SECRET=

DATABASE_URL=

## Base de données fournie

A la racine du dossier : p8vehiloc.sql
